# Launch Exercises

Follow these steps to launch the exercise via the terminal:

1. Open exercise folder in integrated terminal:

   ```bash
   cd path/to/exercise
   ```

2. Install dependencies:

   ```bash
   npm install
   ```

3. Launch project:

   3.1. Start the build:

   ```bash
   npm run build
   ```

   3.2. Start Vite:

   ```bash
   npm run start
   ```

4. Once it starts, check your browser.

That's it! You're ready to go.
